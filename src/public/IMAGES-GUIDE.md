## ✅ الملفات المطلوبة

### 1️⃣ **الأيقونات (Favicons)**

#### `favicon.svg` (أيقونة SVG)
- **الاستخدام**: الأيقونة الأساسية للموقع (تظهر في تاب المتصفح)
- **النوع**: SVG
- **التوصيات**: 
  - ملف شعار الشركة بصيغة SVG
  - يفضل أن يكون مربع (1:1)
  - استخدم اللون الذهبي #EFB621

#### `favicon-16x16.png`
- **الحجم**: 16×16 بكسل
- **النوع**: PNG
- **الاستخدام**: أيقونة صغيرة للمتصفحات القديمة

#### `favicon-32x32.png`
- **الحجم**: 32×32 بكسل
- **النوع**: PNG
- **الاستخدام**: أيقونة متوسطة للمتصفحات

#### `apple-touch-icon.png`
- **الحجم**: 180×180 بكسل
- **النوع**: PNG
- **الاستخدام**: عند حفظ الموقع على شاشة iPhone/iPad

---

### 2️⃣ **شعار الشركة**

#### `logo.png`
- **الاستخدام**: شعار الشركة الرسمي (يظهر في Schema.org ونتائج Google)
- **النوع**: PNG (خلفية شفافة)
- **الحجم الموصى به**: 512×512 بكسل أو أكبر
- **التوصيات**:
  - جودة عالية
  - خلفية شفافة
  - حجم ملف صغير (< 100KB)

---

### 3️⃣ **صورة المشاركة الاجتماعية (OG Image)** ⭐ **مهم جداً!**

#### `og-image.jpg`
- **الحجم الدقيق**: 1200×630 بكسل (نسبة 1.91:1)
- **النوع**: JPG أو PNG
- **الاستخدام**: 
  - تظهر عند مشاركة الموقع على فيسبوك
  - تظهر عند مشاركة الموقع على تويتر
  - تظهر عند مشاركة الموقع على لينكد إن
  - تظهر في رسائل واتساب/تليجرام
  
- **التصميم الموصى به**:
  ```
  ┌─────────────────────────────────────────┐
  │                                         │
  │  [شعار Artal]  Artal Security          │
  │                                         │
  │  Professional Security Services         │
  │  خدمات أمنية احترافية                 │
  │                                         │
  │  🛡️ Protecting Your Assets            │
  │                                         │
  └─────────────────────────────────────────┘
  ```

- **نصائح**:
  - استخدم الألوان الذهبية (#EFB621) والداكنة
  - أضف شعار الشركة بشكل واضح
  - أضف نص قصير يصف الشركة
  - تجنب النصوص الصغيرة (لن تظهر جيداً)
  - حجم الملف: < 300KB

        ├── favicon-32x32.png

        ├── logo.png
        └── og-image.jpg     ← مهم جداً!

## ⚙️ البناء والنشر

عند تنفيذ الأمر:
```bash
npm run build
```

سيتم **نسخ تلقائي** لمحتويات مجلد `public/` إلى مجلد `dist/`:

```
dist/
├── index.html
├── assets/
│   ├── index-xxxxx.js
│   └── index-xxxxx.css
├── images/              ← تم النسخ تلقائياً!
│   ├── favicon.svg
│   ├── favicon-16x16.png
│   ├── favicon-32x32.png
│   ├── apple-touch-icon.png
│   ├── logo.png
│   └── og-image.jpg
├── robots.txt
├── sitemap.xml
└── site.webmanifest
```

---

## 🚀 الرفع على Hostinger

بعد `npm run build`، ارفع محتويات مجلد `dist/` بالكامل:

```
1. افتح File Manager في Hostinger
2. انتقل إلى public_html/
3. ارفع كل محتويات dist/ (بما فيها مجلد images/)
4. تأكد من رفع:
   ✅ index.html
   ✅ مجلد assets/
   ✅ مجلد images/
   ✅ robots.txt
   ✅ sitemap.xml
   ✅ site.webmanifest
```
