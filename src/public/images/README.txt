📁 مجلد الصور والأيقونات
===========================

📌 ضع الملفات التالية في هذا المجلد:

1. favicon.svg          - أيقونة الموقع الرئيسية (SVG)
2. favicon-16x16.png   - أيقونة 16x16 بكسل
3. favicon-32x32.png   - أيقونة 32x32 بكسل
4. apple-touch-icon.png - أيقونة iOS (180x180 بكسل)
5. logo.png            - شعار الشركة (لـ Schema.org)
6. og-image.jpg        - صورة المشاركة الاجتماعية (1200x630 بكسل)

---

✅ كل ما تضعه هنا سيتم نسخه تلقائياً إلى مجلد dist عند تنفيذ:
npm run build

---

📝 ملاحظات:
- og-image.jpg: صورة تظهر عند مشاركة الموقع على فيسبوك/تويتر/لينكد إن
- الحجم المثالي: 1200×630 بكسل
- تأكد من أن الصور بجودة عالية وحجم ملف صغير (ضغط WebP موصى به)
